# Wiki Query Workflow

## When to query

For historical, decision, source-backed, or "what did we decide before?" tasks.

## How to query

1. Read `wiki/index.md` for navigation
2. Read `wiki/log.md` for recent operations
3. Search `wiki/` directories:
   - `wiki/concepts/` — concept explanations
   - `wiki/entities/` — key entities
   - `wiki/syntheses/` — cross-cutting analysis
   - `wiki/questions/` — open questions

4. If not found in wiki, search `raw/` for source material

## Rules

- Search wiki before searching raw transcripts
- Link to wiki pages from memory-bank when referencing durable knowledge
- Do not duplicate wiki content into memory-bank
