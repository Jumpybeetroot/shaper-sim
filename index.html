About the Math & Attribution

This simulator runs a Javascript port of Klipper's internal shaper_calibrate.py math in real-time. The chart shows the maximum acceleration each shaper can sustain before it introduces too much smoothing (defined by the Smoothing Threshold) into your prints.

Attribution: The Input Shaper frequency response, attenuation scaling, and scoring algorithms are based on the source code of Klipper3d, which is licensed under the GNU GPLv3 license.

Predictive Mode

The mechanical predictions are rough estimates based on a simplified mass-spring system $f = \frac{1}{2\pi} \sqrt{K/M}$. Actual results will vary. Always use an ADXL345 accelerometer for precise tuning.
